# Phase 1 — Spine + LLM Core — Plan Index

> **For agentic workers:** Phase 1 is decomposed into milestone plans. Implement them **in order** — each produces working, tested software and is a dependency of the next. Use `superpowers:subagent-driven-development` or `superpowers:executing-plans` per milestone.

**Phase 1 goal:** a standalone, best-in-class LLM gateway — OpenAI/Anthropic/Responses ingress, ~6 Tier-1 providers + passthrough + a hot-reloading model registry, virtual keys with USD budgets and rate limits, exact-match cache, authoritative cost + request logs, all in a single binary with an embedded dashboard that boots zero-config — with the spine invariants enforced and a golden-fixture conformance suite gating every translation.

**Design source:** `docs/2026-06-10-oximy-gateway-design.md` (esp. §2 spine invariants, §4 provider strategy, §5 unified API, §6 lifecycle, §9 performance).

**Exit criteria for the whole phase:** the OpenAI SDK works unchanged against the gateway for all Tier-1 providers (stream + tools); per-request cost matches the registry to the µUSD; budgets hard-block before the upstream call and never overspend under concurrent load; a revoked key stops working immediately; conformance suite green; `oximy-gateway up` boots and serves the dashboard with no config file.

## Conventions for every milestone

- **TDD, no exceptions.** Failing test first; minimal impl; green; commit. Translation/streaming changes require golden-fixture coverage.
- **Money is integer-only.** Never use floats for cost. Unit: **µUSD** (1 USD = 1_000_000 µUSD); prices stored as **i64 µUSD-per-million-tokens**. (Defined in P1.1.)
- **The spine invariants (design §2) are sacred** — fail-closed budgets, single idempotency key across retries, atomic reserve/commit/refund, auth-by-default, cost-correctness. A change that breaks one is a bug even if tests pass.
- **Commands:** `cargo test -p <crate>` per task; `cargo clippy --all-targets -- -D warnings` and `cargo fmt --all` before each commit; commits are DCO-signed (`git commit -s`).
- **Conventional commits:** `feat(spine): …`, `test(llm): …`, `fix(cache): …`.
