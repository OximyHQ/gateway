# NAMING — Top 10 Candidates (Synthesis of rounds 1–3)

Product: open-source AI gateway — unified LLM + MCP, one binary, fastest, agent-first (CLI + own MCP server).

Taste bar applied: short, typable as a CLI command, no cute misspellings, real-word or real-place gravitas — must sound at home next to `envoy`, `caddy`, `vector`. Compounds like ModelPort/OmniRoute cut on taste (product-y, not infra-y). Euripus/Frontinus cut on spell/length friction. Cardea/Claviger cut on registry squats.

## Cut from finals (with reasons)
- **Frontinus** — best governance story (Rome's aqueduct auditor) but 9 letters / 3 syllables fails the CLI-typability bar.
- **Euripus** — flow metaphor is perfect; spelling (yoo-RYE-pus, Euripos variant) is not.
- **Cardea / Claviger** — best mythology, but npm/PyPI squats force suffixed packages from day one.
- **ModelPort / OmniRoute / AgentGate etc.** — descriptive compounds read SaaS-product, not infrastructure; would look wrong next to envoy/caddy/vector.
- **Switchboard / Junction / Patchbay / Viaduct / Gantry / Causeway / Vrata** — live collisions in or adjacent to the gateway/proxy/AI-tooling category (incl. an existing MCP gateway named Switchboard and Apache Causeway trademark).
